Turn retainers into kittens

This is a script that turns all instances of the word "Retainer" on Rule7 into the word "Kitten". There are a number of reasons why I would like to do this, all of them well thought out and obvious.

Greasemonkey is a firefox extension that allows you to add scripts to web sites & pages to make them work better.

  • If you have Greasemonkey installed, you can click here to install Kittens as a greasemonkey script.
  • If you do not, this link will install it as a standalone Firefox extension
